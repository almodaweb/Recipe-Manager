# Recipe-Manager

## Project Description

The **Digital Recipe Manager** is an interactive web application built with **Streamlit** to help the user organize, explore, and plan their recipes efficiently. This app allows users to:
- Add and store new recipes with details like ingredients, preparation time, difficulty, and ratings  
- Search recipes by ingredient  
- View all recipes and track cooking history  
- Get random recipe suggestions  
- Generate combined shopping lists for multiple recipes

## Features

1. **Add a New Recipe:** Input recipe details, including ingredients, category, servings, cooking instructions, preparation time, difficulty, and rating.  
2. **Search by Ingredient:** Filter recipes containing a specific ingredient.  
3. **View All Recipes:** Browse, select, and see recipe details as well as log cooking history.  
4. **Random Recipe Suggestion:** Fetch a random recipe and scale ingredients based on servings.  
5. **Generate Shopping List:** Combine ingredients from multiple recipes into a shopping list.
